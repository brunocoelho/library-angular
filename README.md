## Setup

Se você usa rvm crie sua própria gemset e rode o bundle:

`rvm --create --rvmrc use [RUBY_VERSION]@[GEMSET_NAME]`

`bundle install`

## Testes

Para rodar os testes siga os comandos:

`rake db:test:clone`

`rspec spec/models`

`testacular start spec/javascripts/testacular.conf.js`
